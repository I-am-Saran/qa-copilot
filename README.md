# QA Copilot – AI Powered Bug Triage Agent

## Problem

QA engineers spend significant time manually analyzing defects, assigning severity and priority, identifying root causes, and creating structured bug reports.

## Solution

QA Copilot uses Gemini AI to analyze bug descriptions and automatically generate:

* Component Identification
* Issue Classification
* Severity Analysis
* Priority Analysis
* Bug Type Classification
* Root Cause Analysis
* Test Case Generation

## Tech Stack

* React
* FastAPI
* Gemini 2.5 Flash
* Python

## Features

* AI Bug Report Generation
* Severity & Priority Prediction
* Root Cause Analysis
* Test Case Suggestions
* Structured QA Report Output

## How to Run

### Backend

```bash
cd backend
uvicorn app:app --reload
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```
